Склонение ФИО по падежам для Parser 3

Склоняет русские фамилии, имена и отчества из именительного в другие падежи. Подробнее на GitHub

^untaint{$example.text}

Пример преобразования

$fio[^FioMorph::create[ ^rem{ фамилия } $.surname[Мышкин] ^rem{ имя } $.first[Лев] ^rem{ отчество } $.middle[Николаевич] ^rem{ 1/0 -> мужчина/женщина } $.gender(1) ]] $fio1[^FioMorph::create[ ^rem{ фамилия } $.surname[Сидорова] ^rem{ имя } $.first[Сидора] ^rem{ отчество } $.middle[Сидоровна] ^rem{ 1/0 -> мужчина/женщина } $.gender(0) ]]
Падеж Мужской Женский
Именительный ^fio.fullName[] ^fio1.fullName[]
Родительный (справа) ^fio.fullName[R;R] ^fio1.fullName[R;R]
Дательный ^fio.fullName[dat] ^fio1.fullName[dat]
Винительный ^fio.fullName[accusative] ^fio1.fullName[accusative]
Творительный ^fio.fullName[тв] ^fio1.fullName[тв]
Предложный ^fio.fullName[предложный] ^fio1.fullName[предложный]

Короткая запись Фамилия (родительный) + инициалы слева

^fio.shortName[r;l]

Короткая запись Фамилия (дательный) + инициалы справа

^fio.shortName[дательный]

Фамилия (предложный)

^fio.surName[предложный]

Имя (винительный)

^fio.firstName[V]

Отчество (творительный)

^fio.middleName[ins]

Пол

^fio.gender[full]
